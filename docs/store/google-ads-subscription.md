## Google Play Console 광고/수익화 설정 (Motion Alarm Pocket)

> 2026-05-30 현재 MVP 구현 기준입니다. 이 앱은 무료 + AdMob 광고 모델입니다. 인앱 구매나 구독은 현재 구현되어 있지 않으므로 Play Console에 유료 상품을 등록하지 않습니다.

### 앱 수익 모델
- 기본: 무료
- 광고: AdMob 배너, 자연스러운 지점의 전면/보상형 광고
- 프리미엄/구독: 현재 미구현

### 광고 배치 원칙
- 핵심 기능 시작 전 강제 광고 금지
- 세션 종료, 설정 저장, 힌트 요청, 결과 확인처럼 흐름이 끊기는 지점에만 표시
- 권한 요청 직전/직후에는 광고를 표시하지 않음
- 시스템 경고, 실제 통화, 실제 보안 알림처럼 오해될 수 있는 광고 주변 UI 금지

### 향후 프리미엄 도입 시 후보
- 광고 제거 일회성 구매만 검토
- 구독형 상품은 현재 MVP 범위에서 제외
- 구현 전에는 스토어 설명과 스크린샷에 프리미엄 혜택을 노출하지 않음
