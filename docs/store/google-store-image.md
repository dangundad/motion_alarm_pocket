## 캡처 팁

### 일반 가이드라인

- 해상도: 세로 1080x1920 이상 권장
- 형식: PNG 권장
- 디버그 표시, 테스트 계정, 개인 정보, 실제 전화번호가 노출되지 않게 확인
- 하단 테스트 배너가 CTA나 핵심 상태를 가리지 않는지 확인
- 스토어 이미지에 들어가는 큰 문구는 2줄 이내로 유지한다.

### 강조할 요소

- 표시명: Motion Alarm Pocket / 모션 알람 포켓
- 실제 조작 흐름: 시작, 감시/예약, 알림/결과, 해제 또는 다음 행동
- 로컬 처리와 한계: 기기 센서/마이크/알림 상태에 따라 결과가 달라질 수 있음
- 광고 제거 후원은 실제 Premium 화면이 있는 앱에서만 보여준다.

### 스크린샷 배치 순서

1. 대표 이미지 - 한눈에 보는 모션 알람
2. 시작 지연 - 준비 후 감시
3. 감시 중 - armed 상태
4. 경보와 해제
5. 사운드와 감도 설정
6. Premium - 광고 제거 후원

## 추가 권장사항

### 타겟 사용자

- 짧은 세션으로 바로 확인하고 싶은 사용자
- 앱 안에서 명확한 시작/중지 흐름을 원하는 사용자
- 과장된 보장보다 로컬 유틸리티의 한계를 이해하고 쓰는 사용자

### 핵심 가치 제안

- 앱을 열면 바로 핵심 CTA를 볼 수 있다.
- 센서/마이크/알림 처리는 사용자가 시작한 로컬 흐름을 기준으로 한다.
- 기록과 Premium 안내는 핵심 흐름을 방해하지 않는 위치에 둔다.

### 차별화 요소

- 호기심 훅을 유지하되 실제 기능과 정책 한계를 명확히 분리한다.
- 스크린샷은 실제 에뮬레이터/앱 화면을 근거로 구성한다.
- 광고와 Premium은 기능 보장이 아니라 운영/후원 맥락으로 설명한다.
