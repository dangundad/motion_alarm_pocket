## Google Play 스크린샷 가이드

### 스크린샷 개요

- **목표**: Motion Alarm Pocket이 "폰을 주머니나 가방에 두면 움직임을 감지해 경보를 울리는 간단한 도구"라는 점을 첫 이미지에서 이해시키기
- **핵심 메시지**: 움직임 감지 알람 / 딜레이·감도 설정 / 사이렌+진동 경보 / 감지 기록 / 기기 내 처리(on-device)
- **톤**: 미니멀, 클린, 차분한 실용 도구. 앱 테마(Red M3) 액센트를 살린 라이트/다크 일관 톤
- **주의**: 전문 보안 장비, 도난 방지 보장, 100% 감지, 원격 알림, 위치 추적, 카메라 감시를 암시하지 않음. 엔터테인먼트·일상 유틸리티 톤을 유지

## 스크린샷 구성 (5장)

### 1장: 대표 이미지 - 한눈에 보는 알람

- 제목(KO) 후보: **움직임 알람** / **주머니 알람** / **폰 지킴이**
- 부제목(KO) 후보: **두면 감지, 움직이면 경보** / **시작만 누르면 끝** / **주머니·가방에 그대로**
- Title(EN) 후보: **Motion Alarm** / **Pocket Guard** / **Don't Touch**
- Subtitle(EN) 후보: **Place it, it watches** / **One tap to arm** / **Siren when it moves**

**캡처 화면**

- 홈(`HomePage`)의 상태 원형 인디케이터(Ready/idle 상태)와 Start 버튼이 보이는 상태.
- 앱 정체성(움직임 알람)이 드러나도록 상태 원형 + 메인 액션 버튼을 중심으로 촬영한다.
- 하단 배너 광고가 보이지 않도록 프리미엄/광고 미로드 상태에서 촬영하거나 배너 영역을 제외한다.

### 2장: 설정 - 딜레이와 감도

- 제목(KO) 후보: **딜레이·감도** / **내 맘대로 설정** / **민감도 선택**
- 부제목(KO) 후보: **둘 시간을 주는 딜레이** / **낮음·중간·높음 감도** / **상황에 맞춰 조절**
- Title(EN) 후보: **Delay & Sensitivity** / **Tune It** / **Your Settings**
- Subtitle(EN) 후보: **Delay before arming** / **Low, medium, high** / **Adjust to fit**

**캡처 화면**

- 홈의 설정 카드(`_SettingsCard`) — 딜레이 슬라이더 + 감도 SegmentedButton(낮음/중간/높음)이 보이는 상태.
- 세션 비활성(설정 가능) 상태로 촬영한다.

### 3장: 감시 중 - 움직임을 지켜보는 상태

- 제목(KO) 후보: **감시 중** / **지켜보는 중** / **경계 모드**
- 부제목(KO) 후보: **움직임을 감시합니다** / **폰을 가만히 두세요** / **변화량 실시간 표시**
- Title(EN) 후보: **Armed** / **Watching** / **On Guard**
- Subtitle(EN) 후보: **Watching for motion** / **Keep the phone still** / **Live delta meter**

**캡처 화면**

- 홈의 상태 원형이 armed 상태(원형 progress + 보호 아이콘)인 화면.
- 상태 텍스트("감시 중 – 움직임을 감지합니다")와 변화량(delta) 표시가 함께 보이는 상태.

### 4장: 경보와 기록

- 제목(KO) 후보: **움직임 감지** / **경보 발생** / **감지 기록**
- 부제목(KO) 후보: **사이렌과 진동으로 알림** / **한 번에 경보 중지** / **최근 감지 로컬 기록**
- Title(EN) 후보: **Motion Detected** / **Alarm** / **History**
- Subtitle(EN) 후보: **Siren and vibration** / **One tap to stop** / **Recent detections, local**

**캡처 화면**

- 알람 발생 상태(`alarm_active`, 빨강 원형 + "경보 중지" 버튼) 또는 감지 기록 리스트(`_HistoryTile`) 화면.
- 기록 컷은 1~5개 항목이 시각/날짜와 함께 채워진 상태에서 촬영한다.

### 5장: Premium - 광고 제거 후원

- 제목(KO) 후보: **프리미엄 후원** / **광고 제거** / **앱 응원하기**
- 부제목(KO) 후보: **일회성 후원, 광고 제거** / **3단 후원 옵션** / **한 번 후원, 광고 없이**
- Title(EN) 후보: **Premium Support** / **Remove Ads** / **Support Us**
- Subtitle(EN) 후보: **One-time, ad-free** / **Three support tiers** / **Pay once, no ads**

**캡처 화면**

- Premium 화면(`PremiumPage`).
- Small / Medium / Large 3단 일회성 후원 옵션, 광고 제거 혜택, 하단 구매 버튼이 보이는 상태.
- 구독처럼 보이는 문구를 쓰지 않는다. 모든 옵션은 동일하게 광고를 제거한다.

### 선택 컷

스토어 listing에 여유가 있거나 대체 컷이 필요할 때 사용한다.

- **온보딩**: `OnboardingPage` 첫 스텝 — 알람 아이콘 + "주머니 속 움직임 알람" 가치 소개. 첫인상 강조용.
- **사용 방법 카드**: 홈의 `_HowToUseCard` — 3단계 사용 안내. 사용 흐름 설명용.

## 캡처 팁

### 일반 가이드라인

- 해상도: 세로 1080x1920 이상 권장
- 형식: PNG 권장
- 디바이스: 실제 Android 기기 캡처 권장
- Android edge-to-edge 상태에서 상단/하단 여백이 깨지지 않는지 확인
- 광고가 보이는 컷은 피한다. 프리미엄 컷은 구매 전 화면만 사용한다
- 스토어 이미지에 들어가는 큰 문구는 2줄 이내로 유지한다
- 디버그 표시, 테스트 문구, 실제 개인 메모가 노출되지 않게 확인한다
- 밝은 테마와 좁은 모바일 폭에서 텍스트 겹침이 없는지 확인한다

### 강조할 요소

- 상태 원형 인디케이터의 idle / arming / armed / alarm 색상 계층
- 딜레이 슬라이더와 감도 SegmentedButton
- 사이렌+진동 경보와 "경보 중지" 액션
- 최근 감지 기록(시각 포함)
- 움직임 감지가 기기 안에서만 처리된다는 점

### 스크린샷 배치 순서

1. 대표 이미지 - 한눈에 보는 알람
2. 설정 - 딜레이와 감도
3. 감시 중 - 움직임을 지켜보는 상태
4. 경보와 기록
5. Premium - 광고 제거 후원

## 추가 권장사항

### 타겟 사용자

- 카페·도서관·체육관 등에서 잠시 자리를 비울 때 폰/가방을 지키고 싶은 사용자
- 누가 내 폰을 만지면 바로 알아채고 싶은 사용자
- 별도 권한·계정 없이 기기 안에서만 동작하는 가벼운 알람을 원하는 사용자

### 핵심 가치 제안

- 딜레이로 폰을 둘 시간을 확보한 뒤 시작되는 움직임 감시
- 낮음·중간·높음 감도 선택으로 상황에 맞춘 민감도
- 사이렌+진동 경보와 직접 중지, 최근 감지 로컬 기록
- 위치·계정·별도 권한 없이 기기 안에서만 처리하는 on-device 동작

### 차별화 요소

- 움직임 알람에 단순하게 집중한 흐름 (복잡한 설정·계정 없음)
- 상태 원형 인디케이터로 idle/arming/armed/alarm을 한눈에
- 감지 보장 대신 기기·환경에 따라 달라질 수 있음을 정직하게 안내
- 구독이 아닌 일회성 후원형 프리미엄
