## 이미지 사양
- 해상도: 1080 x 1920 px 권장
- 형식: PNG 또는 JPG
- 장수: 최소 2장, 권장 5장 이상
- 최대 용량: 이미지당 8MB

## 체크리스트
- [ ] 실제 앱 화면만 사용
- [ ] 권한/정책 고지가 필요한 앱은 해당 화면 포함
- [ ] 과장된 보안, 실제 통신, 실제 금/보물 탐지 문구 제외
- [ ] 광고 지점은 사용 흐름이 끝난 화면 기준으로만 표현
- [ ] 밝은 테마와 좁은 모바일 폭에서 텍스트 겹침 확인
